零件繪製

front joint

nx12絞鍊

利用老師所給連結裡的零件使用 nx12 修改 "nx12絞鍊"與 "front joint" 零件的外型

心得:使用NX12來繪製修改零件，運用老師零件檔的草圖進行修改，在修改nx12絞鍊時，因為是左右相反，所以邊看老師所給的草圖邊自行重新繪製一個，在組裝時遇到比較多約束衝突卡住較久，同組組員有幫忙解決問題。

uarm作動影片

心得:組完uarm機械手臂後丟入CoppeliaSim放入joint並排列，第一次用完且放入程式作動後零件會散落，在修改一些參數後回歸正常，只是需要注意不要拉超過機械手臂負荷。

4.測試題目 3. 的 XML UI slider 控制功能

youtube連結:https://youtu.be/EH6xfPyyWbk

心得:使用老師給的壓縮檔裡 uarm_tkinter_control.py 檔案，在CoppeliaSim按下播放鍵後使用SciTE放入程式→Tools→Go，控制三軸作動。

5.uArm 機械手臂場景中, 加入 suctionPad 零件, 以手動操控或程式操控的方式, 錄製 block 方塊零件的取放模擬影片

youtube連結:https://youtu.be/-Rz9K7bykQU

心得:操作像w15一樣使用鍵盤控制joint1、joint2、joint3吸取方塊並可以放置。

6.由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 以跨網路 Python remote API 的方式, 錄製圖形介面 slider 控制前三軸的轉動影片

youtube連結:https://youtu.be/_fgaAbr60-8

心得:在cmd裡先輸入指令 ipconfig /all 查看開啟機械手臂組員 IPV4位址，而後將程式的self.clientID' ..... '中的 IPV4 改成組員的 IPV4 位址 Tools→Go 後按下To connect 連接到機械手臂並完成三軸作動。

零組件檔:nx12
